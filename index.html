Styling in CSS

We can apply styles to various elements using the CSS file. We can style the html element directly:

h1 {
color: #0066FF;
font-size: 2em;
font-family: sans-serif;
}

What if you don't want to change all the h1 elements to blue? What if you want to only change specific parts of the page?

What is a Class?

The html elements have a class attribute which points to a class name in a style sheet. Multiple elements can use the same class name. To define a class in a style sheet write a period (.) character, followed by the name of the class.

.intro {
background-color: yellow;
} 		right-menu
